# Portfolio

Personal portfolio website built with React.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React
- JavaScript
- CSS

## Pages

- **Home** – Introduction and social links
- **About** – Skills and background
- **Projects** – ML and web projects
- **Contact** – Get in touch
